# Stewardship Book Index

This index organizes the manuscript source in `chapters/` into a recommended reading sequence.

## Recommended Reading Order

1. [Introduction](../chapters/00-introduction.md)
2. Read Part I through Part IV in numeric order.
3. Revisit chapters based on the material domain you are facing now: ownership, money, debt, provision, generosity, consumption, body, home, tools, food, time, property, waste, technology, enterprise, inheritance, poverty, crisis, public stewardship, or ecology.
